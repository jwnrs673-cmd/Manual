# 社員司令塔 一日運営マニュアル

## 目的

社員は、売場作業を自分で抱え込む人ではなく、店舗全体を見て「今、誰に、何を、どの順番でやってもらうか」を決める司令塔です。このマニュアルでは、社員が一日を通して確認すべきこと、スタッフへ出す指示、判断基準、引き継ぎ内容を整理します。

## 社員の役割

社員の役割は、作業者ではなく店舗運営のコントローラーです。

- 店舗全体の状況を把握する。
- 優先順位を決める。
- スタッフへ具体的に指示する。
- 作業完了を確認する。
- 異常や遅れを早めに修正する。
- 店長・次の責任者へ必要事項を報告する。

## やらないこと

社員が作業に入りすぎると、店舗全体を見る人がいなくなります。原則として、社員は以下を避けます。

- 長時間の品出しに入り続ける。
- レジに入りっぱなしになる。
- 清掃や返品戻しを自分だけで片づける。
- スタッフに任せられる作業を抱え込む。
- 指示後の完了確認をしない。

ただし、安全、クレーム、事故、重大なレジトラブルなど、社員が直接対応すべき場面は例外です。

## 基本の動き方

社員は一日を通して、次のサイクルを繰り返します。

1. **見る**: 店舗、レジ、売場、バックヤード、人員を確認する。
2. **決める**: 今一番優先すべきことを決める。
3. **指示する**: 誰に、何を、いつまでに、どの状態までやるか伝える。
4. **確認する**: 指示した作業が完了したか見る。
5. **修正する**: 遅れ、混雑、欠品、トラブルがあれば指示を変える。
6. **記録する**: 残作業、異常、引き継ぎを残す。

## 指示の出し方

指示は「何となくお願い」ではなく、具体的に出します。

| 悪い指示 | 良い指示 |
| --- | --- |
| 売場を見ておいて | 10時までに入口平台の前出しと価格表示を確認してください |
| レジお願い | 3番レジを開けて、待ち人数が3人以下になるまで応援してください |
| 品出し進めて | 飲料の特売品を先に2台車分、11時までに平台へ出してください |
| きれいにして | 通路の段ボールを撤去し、床に商品がない状態にしてください |

## 一日のタイムライン

| 時間帯 | 社員が確認すること | スタッフへ出す主な指示 | 完了確認 |
| --- | --- | --- | --- |
| 朝・開店前 | 人員、外周、レジ、売場、特売、連絡事項 | 開店準備、外周確認、レジ準備、重点売場確認 | 開店できる状態か |
| 午前 | 入荷量、欠品、品出し進捗、レジ待ち | 検品、優先品出し、欠品補充、レジ応援 | 優先商品が売場に出ているか |
| 昼前後 | 休憩回し、レジ混雑、売場乱れ | 休憩順、レジ応援、カゴ・カート補充 | ピークに耐えられる体制か |
| 夕方前 | 午前残作業、売場回復、夕方売れ筋 | 前出し、返品戻し、売れ筋補充、清掃 | 夕方ピーク前に売場が整っているか |
| 夕方ピーク | レジ待ち、問い合わせ、通路安全、欠品 | レジ応援、商品案内、通路整理、重点補充 | 混雑と危険が放置されていないか |
| 夜・閉店前 | 残作業、レジ締め準備、売場最終状態 | 返品戻し、ゴミ回収、閉店準備、翌日準備 | 閉店後作業が明確か |
| 閉店後 | レジ締め、施錠、防犯、引き継ぎ | 最終巡回、施錠確認、記録作成 | 金銭・安全・防犯が完了したか |

## 朝・開店前に社員がすること

### 確認すること

- 今日の出勤者、欠勤、遅刻、早退予定
- 新人や不慣れなスタッフの有無
- 店舗外周、入口、駐車場、カート置場の異常
- レジ台数、釣銭、決済端末、レジ備品
- チラシ、特売、重点商品、売価変更、POP
- 前日からの残作業、クレーム、設備不良

### 指示すること

| 指示先 | 指示内容 |
| --- | --- |
| レジ担当 | 開店前にレジ・釣銭・決済端末・袋・レジロールを確認する |
| 売場担当 | 入口平台、特売品、欠品しやすい商品、価格表示を確認する |
| 外周担当 | 入口、駐車場、カート置場、ゴミ、危険物を確認する |
| 新人スタッフ | 今日の担当範囲と困った時の報告先を伝える |

### 朝礼で伝えること

- 今日の売上目標または重点方針
- 特売商品、売価変更、注意商品
- レジ応援の基準
- 休憩予定
- 各スタッフの担当範囲
- 前日からの引き継ぎ事項

### 朝の完了基準

- 開店できない理由がない。
- 誰が何を担当するか明確である。
- 特売品と価格表示に大きなズレがない。
- レジが開店直後から使用できる。

## 午前に社員がすること

### 確認すること

- 入荷量と検品の進み具合
- バックヤードの滞留商品
- 欠品しやすい商品
- 特売品と売れ筋商品の売場在庫
- レジ待ち人数
- 通路に台車や段ボールが出っぱなしになっていないか

### 指示すること

| 状況 | 指示例 |
| --- | --- |
| 入荷が多い | 検品担当と品出し担当を分け、特売品から先に出す |
| 欠品がある | 担当者へ在庫確認と補充を指示する |
| レジが混み始めた | 品出しを一時停止し、レジ応援に入るスタッフを指定する |
| バックヤードが詰まっている | 通路確保、優先品出し、保管場所整理を指示する |

### 午前の完了基準

- 優先度の高い商品が売場に出ている。
- レジ混雑が放置されていない。
- バックヤードと売場通路が危険な状態ではない。

## 昼前後に社員がすること

### 確認すること

- レジの待ち人数
- スタッフの休憩順と売場の人数
- カゴ、カート、袋、レジロールの残量
- 入口、レジ前、特売平台の乱れ
- 問い合わせが増えている商品の場所

### 指示すること

| 状況 | 指示例 |
| --- | --- |
| 休憩で人が減る | レジ担当を残し、売場担当の休憩順をずらす |
| レジ待ちが増える | 応援者を指定し、待ち人数が落ち着くまで対応させる |
| カゴ・カートが不足 | 入口補充担当を決め、ピーク前に補充させる |
| 売場が乱れている | 短時間で入口・特売・日配など重点売場だけ整えさせる |

### 昼の完了基準

- 休憩中でもレジと売場が回る人員配置になっている。
- ピーク前にカゴ・カート・レジ備品が不足していない。

## 夕方前に社員がすること

### 確認すること

- 午前中の未完了作業
- 夕方に売れやすい商品の在庫
- 入口、平台、レジ前、通路の乱れ
- 返品戻し、買い回り商品、放置商品の量
- 清掃が必要な場所

### 指示すること

| 状況 | 指示例 |
| --- | --- |
| 残作業が多い | 今日中に終える作業と翌日に回す作業を分ける |
| 夕方売れ筋が薄い | 飲料、食品、日用品など優先補充を指示する |
| 売場が乱れている | 前出し、フェイスアップ、返品戻しを担当別に割り振る |
| 清掃が必要 | 入口、トイレ、レジ前などお客様接点を優先させる |

### 夕方前の完了基準

- 夕方ピークに入る前に、重点売場とレジ周辺が整っている。
- 残作業の優先順位と担当者が決まっている。

## 夕方ピークに社員がすること

### 確認すること

- レジ待ち人数
- 問い合わせやクレームの発生
- 通路の安全
- 売れ筋商品の欠品
- スタッフが一つの作業に偏っていないか

### 指示すること

| 状況 | 指示例 |
| --- | --- |
| レジ待ちが多い | すぐに応援者を指名し、品出しを止めてレジへ回す |
| 問い合わせが多い | 商品場所を共有し、案内担当を決める |
| 通路が危険 | 台車、段ボール、放置商品を撤去させる |
| 欠品が出ている | 売れ筋だけに絞って補充させる |

### 夕方ピークの完了基準

- レジ待ち、クレーム、通路危険が放置されていない。
- 社員が全体を見て、必要な応援指示を出せている。

## 夜・閉店前に社員がすること

### 確認すること

- レジ締め準備の状況
- 売場の返品戻し、ゴミ、放置商品
- 翌日に残す作業
- 閉店後の担当割り
- 施錠、防犯、最終巡回の担当者

### 指示すること

| 指示先 | 指示内容 |
| --- | --- |
| レジ担当 | レジ周辺備品、カゴ、袋、忘れ物を確認する |
| 売場担当 | 返品戻し、前出し、ゴミ回収を行う |
| 閉店担当 | 最終巡回、施錠、防犯設定の担当範囲を確認する |
| 次日担当への引き継ぎ者 | 残作業、欠品、異常、クレームを記録する |

### 夜の完了基準

- 閉店後に何を誰が行うか決まっている。
- 翌日に影響する残作業が記録されている。

## 閉店後に社員がすること

### 確認すること

- レジ締めと違算
- 売上金、釣銭、金庫管理
- お客様の残留がないか
- 電源、火気、冷蔵冷凍設備
- 入口、非常口、バックヤード、事務所の施錠
- 防犯設定
- 業務連絡ノートまたは共有チャットへの記録

### 閉店後の完了基準

- 金銭確認が完了している。
- 施錠と防犯設定が完了している。
- 残作業と異常が記録されている。

## 社員の巡回チェック

社員は作業に入り続けず、最低でも次の間隔で店舗を一周します。

| 時間帯 | 巡回頻度 | 見る場所 |
| --- | --- | --- |
| 開店前 | 開店30分前と開店直前 | 外周、入口、レジ、重点売場 |
| 午前 | 30〜60分に1回 | 入荷場所、バックヤード、特売売場、レジ |
| 昼前後 | 15〜30分に1回 | レジ、入口、カゴ・カート、特売平台 |
| 夕方ピーク | 15分に1回 | レジ、通路、入口、売れ筋売場 |
| 閉店前 | 閉店60分前、30分前、閉店直前 | 売場、レジ、返品、ゴミ、施錠箇所 |

## 優先順位の判断基準

| 状況 | 優先順位 | 社員の判断 |
| --- | --- | --- |
| 床濡れ、破損、事故 | 最優先 | すぐ作業を止めて安全確保を指示する |
| レジ待ちが多い | 高 | 品出しよりレジ応援を優先する |
| クレーム発生 | 高 | 社員または責任者が直接対応する |
| 特売品の欠品 | 中 | 売場担当へ在庫確認と補充を指示する |
| 通常品の品出し遅れ | 中〜低 | ピーク状況を見て後回しにする |
| 清掃・整理 | 状況次第 | お客様接点を優先する |

## 指示後の確認ルール

指示は出して終わりではありません。社員は必ず結果を確認します。

- 指示した時間までに終わったか。
- 完了基準を満たしているか。
- 途中で困っていないか。
- 他の作業に悪影響が出ていないか。
- 終わらない場合、応援を付けるか、優先順位を変えるか。

## 引き継ぎテンプレート

| 項目 | 記入内容 |
| --- | --- |
| 残作業 | 何が、どこに、どれくらい残っているか |
| 売場状況 | 欠品、価格表示、破損、売れ筋の状況 |
| レジ状況 | 違算、端末不具合、混雑傾向 |
| お客様対応 | クレーム、問い合わせ、取り寄せ、忘れ物 |
| 人員状況 | 欠勤、遅刻、早退、新人フォロー |
| 設備・安全 | 故障、床濡れ、什器破損、防犯上の注意 |
| 次にやること | 次の責任者が最初に確認すべきこと |

## スライド化メモ

- 主メッセージ: 社員は「作業者」ではなく「司令塔」である。
- 図解案: 店舗全体を見て、スタッフへ指示を出す社員の位置づけ。
- 研修クイズ: レジ混雑、入荷遅れ、売場乱れが同時に起きた場合、最初に指示することは何か。
- ロールプレイ案: 夕方ピーク中にスタッフ3名へ優先順位をつけて指示する練習。
