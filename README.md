# ディスカウントストア オペレーションマニュアル

このリポジトリは、小売店舗（ディスカウントストア）の業務手順を収集・整理し、最終的にスライド化できるマニュアルとして管理するための作業場所です。

## 作成方針

1. **原稿を集める**: 既存メモ、業務手順、写真、チェックリストを `01_source_manuals/` に投入します。
2. **構成案に分解する**: 店舗運営の流れ・役割・教育単位ごとに章立てします。
3. **Manual化する**: `02_operations/` と `03_roles_training/` に、現場で使える手順書として整形します。
4. **スライド化する**: `05_slide_deck/` に、研修・朝礼・OJTで使えるスライド原稿へ変換します。

## フォルダ構成

| フォルダ | 用途 |
| --- | --- |
| `00_strategy/` | マニュアル全体の設計方針、章立て、用語集、更新ルール |
| `01_source_manuals/` | ユーザーが投入する元資料・未整理メモ・写真・既存マニュアル |
| `02_operations/` | 店舗オペレーション別の完成マニュアル |
| `03_roles_training/` | 職位別・新人教育・OJT・評価チェックリスト |
| `04_templates/` | 手順書、チェックリスト、スライド原稿のテンプレート |
| `05_slide_deck/` | スライド化するための章別アウトライン・発表者ノート |
| `99_archive/` | 旧版・廃止版・参考資料 |

## 推奨ワークフロー

1. `01_source_manuals/` に作りたいManualの原稿を追加する。
2. `00_strategy/manual_structure.md` に章立てと優先順位を反映する。
3. `04_templates/procedure_template.md` を使って各業務の手順書を作る。
4. `04_templates/checklist_template.md` を使って開店・閉店・清掃などの確認表を作る。
5. `05_slide_deck/slide_outline_template.md` を使って研修用スライド原稿に変換する。

## 命名ルール

- ファイル名は `NN_業務名.md` の形式を推奨します。
- 例: `01_開店準備.md`, `02_入荷検品.md`, `03_レジ対応.md`
- 更新時は本文末尾の「改訂履歴」に日付・変更者・変更内容を記録します。
